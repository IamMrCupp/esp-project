The website uses the following technology:
- bootstrap
- fontawesome
- npm
- browser-sync
- gulp
- dart sass
  
Initial Setup:
``` 
npm install browser-sync gulp sass gulp-sass --save-dev
npm install bootstrap --save
npm install @fortawesome/fontawesome-free --save 
```
